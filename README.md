#### Node (16 LTS) e Yarn

###### Guia de instalação das ferramentas

```bash
node -v
yarn -v
```

Caso precise instalar o Yarn, faça o comando:

```
npm install --global yarn
```



```
yarn create react-app frontend --template typescript
```
OU:
```
npx create-react-app frontend --template typescript
```
